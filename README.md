# JWick-lzh.github.io

个人作品集网站 - 展示自己的一些过往

## 项目简介

这是一个使用现代前端技术栈构建的个人作品集网站，用于展示个人技能、项目经验和联系方式。

## 技术栈

- **框架**: React 19.2.0 + TypeScript
- **构建工具**: Vite 7.2.4
- **样式**: Tailwind CSS + tailwindcss-animate
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **动画**: GSAP
- **表单**: React Hook Form + Zod
- **图标**: Lucide React

## 功能模块

- 🏠 **Hero** - 首页介绍
- 👤 **About** - 个人简介
- 💪 **Skills** - 技能展示
- 💼 **Experience** - 工作经历
- 🎨 **Works** - 项目作品
- 🛠️ **Services** - 服务内容
- 💬 **Testimonials** - 客户评价
- 📧 **Contact** - 联系方式

## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview

# 代码检查
npm run lint
```

## 部署

项目可以部署到 GitHub Pages 或其他静态网站托管服务。
