## 一句话

旧版是"文字介绍牙刷"，新版是"牙刷和文字同台演出"——暗场空间里一段连续镜头旅程，文案全部沿用，骨架推倒重造。

## 定稿决策

- 视觉：全面转暗色沉浸式
- 结构：连续镜头旅程（GSAP ScrollTrigger scrub），弃版块吸附
- 元素：全套——loader、自定义光标、微尘粒子、大字排版+文字动效
- 3D：牙刷模型重建，写实向（植毛、铝材质、暗场布光）
- 内容：7 章文案全部沿用现有 index.html 的文案，不改卖点
- 依赖：本地 gsap.min.js + ScrollTrigger.min.js + three.min.js，零构建，GitHub Pages 直接部署

## 视觉规范

- 底色：深空黑 `#06070a → #0c0e14` 微渐变；文字近白 `#f2f3f5`
- 主点缀：冰蓝 `#6ee7ff`；辅点缀：香槟金（少量线条，延续品牌）
- 巨字：weight 800，紧字距，**字号 = min(20vw, 88vw ÷ 字数)**，永不溢出
- 质感：暗色噪点 grain（反色）、冰蓝辉光渐变、细线分隔

## 空间层级（核心手法）

```
最底层  巨型描边字（章节大字，随滚动视差，最底 z）
中间层  3D 牙刷（穿插于大字笔画间，产生前后遮挡）
最上层  小号信息文字 + UI（卖点、数据、按钮）
```

## 3D 牙刷重建

- 机身：细长椭圆截面手柄（非纯圆柱），阳极氧化铝 PBR（金属度/粗糙度/清漆层写实），正面微凹智能屏 + 独立电源键
- 刷头：带角度窄颈 + instancing 植毛数百簇，白毛为主、蓝指示毛点缀，毛尖高度随机起伏
- 光照：暗场三点布光 + 程序生成暗色影棚环境贴图（反射见柔光条），底部接触阴影，屏幕自发光光晕
- 细节：镀铬驱动轴、Type-C 充电口、底部防滑圈
- 双色沿用：太空灰 / 冰莓粉

## 滚动系统

- 总长：7 章 × 200vh = 1400vh，一条全局 timeline，scrub 平滑 ~0.8
- 每章内部：进场 0–25% → 驻留 25–75%（平台期，画面静止稳定）→ 退场 75–100%
- **snap：停止滚动后 0.6s 吸附到最近章节驻留段中心**，静止时永远是完整画面
- 章节导航轨：左侧固定 `01 旗舰 / 02 工艺 / 03 双色 / 04 拆解 / 05 方案 / 06 实证 / 07 入手`，当前高亮，点击平滑滚到该章 snap 点；右下角滚动进度百分比

## 章节脚本（文案沿用旧版对应版块）

| 章 | 进场 | 驻留（snap 点） | 退场 |
|---|---|---|---|
| 01 旗舰 | loader 揭幕后巨字「智能牙刷」逐字浮现，牙刷从暗处亮起 | 缓慢自转+屏亮，三行卖点就位，CTA 可点 | 巨字视差上移淡出，镜头推近 |
| 02 工艺 | 镜头贴近铝壳，光带扫过一次（只扫一次） | 微距静止，IPX8/智能屏两数据点悬浮 | 镜头绕到正面 |
| 03 双色 | 「灰/粉」半字入场，色块升起 | 正面平视，点击色块→光泽扫过换色（可反复） | 色块下沉，镜头拉开 |
| 04 拆解 | 爆炸度 0→1 由该章前 60% scrub | 后 40% 全展开静止，4 条标注线按爆炸进度依次点亮 | 下章进场时收拢 |
| 05 方案 | 镜头俯冲屏幕特写，四卡从底部升入 | 静止，hover/点击卡片→屏幕 UI 切模式（默认第一张） | 卡片下沉，镜头急速拉远 |
| 06 实证 | 「99.1%」数字滚动（触发一次） | 巨数静止（总宽≤80vw），三数据环绕+信任徽章 | 数字淡出，牙刷回中 |
| 07 入手 | 收尾文案逐行入场，辉光渐起 | 呼吸辉光+缓转，CTA 可点，到底无更多内容 | 无（终点） |

## 屏内约束（红线）

- `html, body { overflow-x: hidden }`；文字入场全部在行遮罩内做，不在屏外绘制
- 每章内容框 `max-width 88vw / max-height 84dvh`；用 `dvh` 避开 iOS 地址栏
- 视差振幅上限 ±4vw / ±3vh
- 拆解标注：文字 clamp 在 6vw 安全区内，仅线条可近边
- <760px：章节轨收成右侧细进度条，05 四卡改 2×2
- `vh < 600`：排版整体缩 85%

## 招牌元素

- Loader：百分比数字+细进度线，跟踪真实加载，完成揭幕
- 光标：圆点+延迟外环，CTA/色块磁吸放大，触屏关闭
- 粒子：暗场漂浮微尘，随滚动速度轻微拖尾、随鼠标微扰
- 文字动效：标题逐行分割上移入场（手写分行，不用付费 SplitText），滚动视差

## 兜底

- `prefers-reduced-motion`：关 scrub 运镜和粒子，改章节淡入淡出，snap 保留
- no-WebGL：降级页保留，按暗色重做样式
- resize/转屏：ScrollTrigger.refresh + 巨字重算字号 + 相机比例更新
- 移动端：降 DPR（≤2），关光标/磁吸，保帧率

## 验收标准

1. 任何视口宽度无横向滚动条，任何章节内容不出屏
2. 停止滚动后总是吸附到某章驻留点，无半截画面
3. 03 换色、05 切模式、06 数字滚动、04 标注点亮全部按触发表工作
4. loader 4s 内必然放行；no-WebGL 显示暗色降级页
5. reduced-motion 下无运镜无粒子仍可读完整内容
